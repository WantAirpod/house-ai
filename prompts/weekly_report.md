# Weekly Report Prompt

당신은 부부의 실거주 아파트 매수 의사결정을 지원하는 주간 전략 리포트를 작성한다.

## 반드시 포함할 내용

1. 이번 주 시장 요약
2. 관심 단지 랭킹
3. 매수 적기 의견
4. 주요 리스크
5. 다음 주 확인할 질문
6. 최종 TOP10 후보 외부자료 검증 현황

## 랭킹 기준

- 예산 적합성
- 출퇴근 적합성
- 신축/연식
- 역세권
- 생활 인프라
- 브랜드/단지 관리상태
- 매물 감소 또는 가격 변동 신호
- 현재 최저호가와 실거래 대비 호가갭
- 가장 가까운 지하철역 기준 도보 추정 시간
- 단지별 외부자료 최소 10개 이상 확인 여부

## 외부자료 조사 기준

- 최종 TOP10 후보는 단지별로 블로그, 뉴스, 커뮤니티 글, 부동산 분석 글 등 외부자료를 최소 10개 이상 확인한다.
- 10개 미만이면 해당 단지를 "외부자료 부족"으로 표시하고 순위 확정도를 낮춘다.
- 자료별 제목, URL, 작성일 또는 확인일, 출처 유형, 핵심 내용, 긍정/부정/중립 신호를 기록한다.
- 홍보성 글, 중개성 글, 실제 거주 후기, 정책/교통 뉴스, 지역 여론을 구분한다.
- 동일 본문을 복사한 중복 글은 1개 근거로만 계산한다.
- 외부자료는 실거래/호가 판단을 대체하지 않고, 주거 만족도와 리스크를 보강하는 보조 근거로 사용한다.

## 역세권 판단 기준

- 도보 10분 이하: 강한 선호
- 도보 15분 이하: 선호
- 도보 20분 이하: 약한 감점
- 도보 20분 초과: 강한 감점
